CSS中的两端对齐

发表于 2015-08-28   |  

最近看了几篇文章都是讲两端对齐，有 justify 的，也有 flexbox 的，觉得都很有用，但由于有些实现起来还是有些限制，如需要特定的结构等，故有必要记录一下。

一、 display: inline-block + text-align: justify

html:

css:

此方案兼容所有浏览器(包括IE6)，主要用于 PC 端，但有一定的限制：

  1. 列表元素间不能没有空隙，最少要有一个空格
  2. 列表结束标签与最后一个元素之间要连接在一起，或者空多一行
  3. 为了兼容 IE67，列表元素需用 inline 元素
  4. 列表个数也比较难控制

最后一行要左对齐

理论上，最后一行都是要左对齐的，在两端对齐的列表里最后一行想处理为左对齐，是很麻烦的，需要最后一行有多少个元素，就加多少个 修复 元素，如下图：
